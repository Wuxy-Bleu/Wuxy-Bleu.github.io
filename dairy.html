DevDairy

Posted on Sat, Aug 17, 2019

2020 7、8月

  • 7月初刚拿到新电脑，上手下，安装下必备的环境。
  • 7月主要将好久没动的SSM的知识拿出来再看看，github上找了些ssm项目查了查，自己整合了下框架搞开发了一个增删改查的。
  • 之后，为了找工作，Java se，网络，数据库等等 整理了些笔记，刷了些面试题，。

8.28

  • 安装了nodejs ，首次接触到了Node出现后的前端世界，各种js框架，vue, react, augular
  • 开始学习Vue

9月

  • 大部分时间花在了学习 vue + es6
  • 折腾自己的开发环境，虚拟机，windows terminal终端，阿里云环境
  • 这个月开始正式工作了

10月

  • 基本是学习vue+js，折腾vscode的一个月。
  • 十一假期，旧电脑上物理安装了manjaro，折腾了一周多

10.29

  • 开始学习springboot

11月

  • 渐渐熟悉了springboot，整合了mybatis，druid，swagger，springmvc越来越熟练了，springboot+mybatis的各种增删改查操作也熟练了
  • 备案了自己的域名，阿里云的ecs的环境慢慢折腾得适合开发了，特别是vim，zsh安装了各种插件，Mysql，java, redis，nginx, nacos, ……一堆的环境都配置好了 加上自己的虚拟机的折腾，linux的操作我也越来越熟练了。
  • idea也折腾了更加好用了，用起来更舒服了，装了不少插件玩
  • github上看了不少springboot的练手项目，接触到了人人开源，悟空cms，wordpress各种东西
  • 学习了docker，但是后来很少用到它

12.1

  • 开始开发这个音乐网站了
  • 中间为了改造 vue-element-admin这个项目花了半个多月一点一点的看他vue代码

12月

  • 开发自己的网站的一个月 + 按自己的需求改造vue-element-admin
  • springboot的springmvc的大部分操作+增删改查那一套熟的不能再熟悉了
  • 最大的成果就是实现了前后端联动，上传文件到oss中
  • 接触了阿里云的更多的功能，oss, cdn, dns等等等等……
  • springcloud我花了大量的时间学习和用到自己的项目中，但是很多时间存在很多问题无法解决，同时慢慢发现，我的项目根本用不到springcloud的业务

2021 1月

  • 所有的笔记迁移到notion中，我折腾了下这个notion，非常的惊喜，很好用
  • 这个月大部分的时间都花在了music_admin模块整合spring security + JWT + redis，实现单点登录的功能
  • 解决了所有的跨域问题，所有的请求都通过nginx统一转发
  • 做了不少的小功能，aop + 注解实现了保存部分请求的详细信息到mongodb中。整合了邮件功能，尝试开发一个websocket聊天室但是失败了
  • 尝试将所有的代码部署到了阿里云上，jar包是直接后台启动的，vue部署到Nginx上。

2021 2月 新年

  • 这个月我再github上找了一个简易的在线音乐网站，附带了完整的前后端和数据库系统，我完全重写了它的后端，不过也很快，半个月不到就基本搞定了。
  • 同时接入了我写好的登录系统，在它的vue代码上加上了全局路由守卫。
  • 后端改造为多数据源，因为我没有抛弃它本身的那套数据库，那么多音乐的资料删掉可惜了。
  • 当你的前端学习理解越来越好的情况下，慢慢发现了前端世界的庞大精彩和复杂。我见识到了各种各样的脚手架，生成一个blog的，生成文档的，生成简历的，各种各样的东西。
  • 这个月我开始跟着 算法第四版 橙书学习算法，同时也是强迫自己多写java代码，而不是总是在框架上开发，同步着去leetcode找些题做做
  • 花了不少时间在netflix spotify一些流媒体上，真的容易沉迷啊😂

三月

  • 自己的项目虽然并没有完善好，但是先做出一个总结，开发整理了这个博客。
  • 折腾了不少新的东西，github.io github profile，前端的技术栈 site generator + headless cms
  • 开发了静态网站丢到了部署在了netflity，初步接触到了前端的 dev-ops 云部署+cdn加速 + dns ....
  • 主动去了解了一些开源社区
  • 对于整个的开发的世界的理解慢慢更加深入了。现在对我来说，接触并开始一个新的技术更加的得心应手了，英文能力更加了，学习的效率比起几个月之前大幅上涨了。
  • 学习了解一点简单的设计模式。
